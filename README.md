# 🎬 Maven Movies SQL Analysis
SQL-based analysis of the Maven Movies database using joins, aggregate functions, CTEs, subqueries, and window functions to uncover insights about customers, rentals, revenue, staff performance, and business trends.

---

## 📌 Project Overview

This project analyzes the Maven Movies database using SQL to generate business insights for a fictional DVD rental company. The analysis focuses on customer behavior, movie performance, rental trends, revenue, and inventory management.

The project demonstrates practical SQL skills including joins, aggregate functions, subqueries, Common Table Expressions (CTEs), window functions, and business reporting.

---

## 📊 Objectives

- Analyze customer rental behavior
- Identify top-performing films
- Calculate revenue insights
- Analyze inventory and rental trends
- Generate business recommendations using SQL
